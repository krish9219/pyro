# Pyro Language Support

Syntax highlighting, bracket matching, and auto-indentation for the [Pyro programming language](https://aravindlabs.tech/pyro-lang/).

## Features

- Syntax highlighting for `.ro` files
- 23 keyword highlighting (fn, let, mut, if, else, for, in, while, return, import, etc.)
- String interpolation `{var}` highlighting
- Comment highlighting (`#`)
- Bracket matching and auto-indentation
- Pipe operator `|>` highlighting
- Lambda `|x|` highlighting

## Quick Start

1. Install this extension
2. Open any `.ro` file
3. Start coding in Pyro!

## About Pyro

Pyro is a compiled language that transpiles to C++20. 79x faster than Python, 23 keywords, 76 built-in modules.

- **Website:** [aravindlabs.tech/pyro-lang](https://aravindlabs.tech/pyro-lang/)
- **Playground:** [Try online](https://aravindlabs.tech/pyro-lang/playground.html)
- **Install:** `irm https://aravindlabs.tech/pyro-lang/install.ps1 | iex` (Windows)
